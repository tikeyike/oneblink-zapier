# oneblink-zapier

Edit the `.blinkmrc.json` file with the following details

- `WEB_HOOK_SECRET`
- `WEB_HOOK_ENDPOINT`
- `FORMS_ACCESS_KEY`
- `FORMS_SECRET_KEY`

Login using the OneBlink CLI `oneblink login`

Set your scope `oneblink api scope <URL GOES HERE>`

Deploy your code `oneblink api deploy`
